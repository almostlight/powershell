## Contributing

To improve these scripts:
1. Fork the repository
2. Make changes
3. Test on a (relatively) clean Windows installation
4. Submit a pull request

Thanks :)

##  Support

- **Issues**: [GitHub Issues](https://github.com/almostlight/powershell/issues)
